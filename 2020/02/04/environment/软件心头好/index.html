软件硬件网站心头好&使用记录

[TOC]

下载网址

网站-系统总裁

下一些装机工具，比如WinPE、驱动程序、封闭程序等

网站-下载之家

直接就能点到链接下东西

网站-潮下载

基础工具类

压缩软件-bandizip

bandizip在windows上天下第一

压缩软件-eZip

mac，想用还没用过

压缩软件-The UnArchiver

mac，能解压，不能压缩，不能预览，免费，效果一般

NTFS-Tuxera NTFS

mac上读U盘的必备工具

系统工具类

WinPE制作-WePE

Windows上可以给U盘装WinPE的轻量极简工具

U盘启动工具制作-DiskMaker

专门用于Mac的U盘启动工具

驱动软件-驱动总裁

离线安装所有驱动

文件搜索工具-Everything

Windows上的本地文件搜索神器

文件搜索工具-Listary

Windows上的本地文件搜索神器

清理优化-Dism++

Windows。可以找到C盘的大文件

杀毒-火绒安全

Windows

清理注册表-CCleaner

Windows

换字体-字由

Windows，不用安装字体

学习类

笔记-GitNote

一个很有潜力的软件，全平台。用的是笔记+git。但目前bug比较多

Markdown笔记-Typora

数学公式-Mathpix

实用工具类

Microsoft Office

必选

PDF-PDF Expert

pdf工具，轻量，收费

PDF-SumatraPDF/pdf x-change reader（更喜欢这个）

云盘-百度云盘

笔记-印象笔记

图片查看-HoneyView

Windows，简洁无广告

远程控制-TeamViewer

好用，非商用可免费，全平台

ssh远程控制-MobaXterm

清单工具-TickTick(滴答清单)

好用，免费版足够使用，全平台

虚拟机-VMware

虚拟机-Parallels

Mac和Windows可用，效果比较好，收费

下载工具-qBittorrent

磁力下载工具，说是比迅雷好用

下载工具-Thunder(迅雷)

没得选，通用

下载工具-PanDownload

可下载百度网盘的内容，适度使用，容易封号

用过Motrix，体验一般

视频播放-爱奇艺本地万能播放器

windows，确实啥都能放，品质不错。就是每次播放都会弹出一个框提示你比较烦

视频播放-Potplayer

Windows
很简洁的视频播放器，启动速度快、解码能力强、耗内存低、无广告、不流氓，最大的特点是倍速播放也不会变音

视频播放-IINA

Mac，没得说的，小巧实用，免费，github上开源

超高清屏幕录制工具-Bandicam

Windows
号称世界三大视频录制神器之一，支持H.264预制高清，可以录制4K的高画质144FPS视频。录制的时候还可以进行截图，添加水印图片，鼠标点击效果等操作！

（免费版好像只能录制十分钟）

学术研究类

文献管理-EndNote

Mac和Windows可用，收费

公式编辑-MathType

Mac和Windows可用，收费
之前我一直觉得MathType不行，后来发现居然有Latex编辑方式，立马觉得很OK

公式识别-MathPix

能识别手写公式或图像类型的公式，超棒
全平台

休闲娱乐类

音乐-网易云音乐

版权不足，很多歌没有，心动模式不错，和B站联系紧密

开发编程类

Git图形界面-Fork

windows和mac

全能-VScode

全能选手，就算当个记事本都很强。就是老是提醒更新，收费

原来喜欢sublime，现在觉得VSCode比sublime强出几条街

Xcode

Mac上的，虽然Xcode不是很好用，但装了Xcode其它很多小东西不用安装了比较方便

C++-CLion

Mac+Windows，在Windows上比Codeblocks强很多。Codeblocks太容易崩了

Python-PyCharm

PyCharm的话有时会出现无法运行的情况，本质是编译器设置的问题。
需要把编译器设置一下。这里设置我老出错的一个点是设置system interpreter，而不是conda/虚拟环境等的inerpreter

Python-Jupyter Lab

Java-IDEA

Web-webStorm

Git-fork
