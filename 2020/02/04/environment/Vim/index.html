插件版

下插件，一件安装
github-ma6174/vim-deprecated 开源代码

包括功能：

  1. 按F5可以直接编译并执行C、C++、java代码以及执行shell脚本，按“F8”可进行C、C++代码的调试
  2. 自动插入文件头 ，新建C、C++源文件时自动插入表头：包括文件名、作者、联系方式、建立时间等，读者可根据需求自行更改
  3. 映射“Ctrl + A”为全选并复制快捷键，方便复制代码
  4. 按“F2”可以直接消除代码中的空行
  5. “F3”可列出当前目录文件，打开树状文件目录
  6. 支持鼠标选择、方向键移动
  7. 代码高亮，自动缩进，显示行号，显示状态行
  8. 按“Ctrl + P”可自动补全
  9. []、{}、()、””、’ ‘等都自动补全

终端执行：

命令执行后，还会继续安装相关插件，但是在使用后来在使用vi编辑python代码时，会先出现一个错误

Error: the pyflakes.vim plugin requires Vim to be compiled with +python

其中pyflakes.vim是一个非常好用的python高亮vim插件，在使用vim编写python代码的时候，出现了需要编译的的错误，解决方法有两种。
方法一：去github重新下载一个pyflakes.vim，将源文件进行替换即可，然后进入git克隆文件目录./pyflakes-vim/ftplugin，将python目录下的所有文件复制到~/.vim/ftplugin目录下即可。复制完成后，再用vim打开python源码文件就不会出现上述错误了。

方法二： 编辑pyflakes.vim将第28行注释掉，即将28行前面添加双引号即可。

Vim使用

复制 y+p

yy复制一行
yw复制当前光标所在位置到单词尾字符的内容到vi缓存区，相当于复制一个单词　　
y$复制光标所在位置到行尾内容到缓存区　　
y^复制光标所在位置到行首内容到缓存区　　
#yy例如：5yy就是复制5行　　
#yw例如：2yw就是复制两个单词　　
如果要复制第m行到第n行之间的内容，可以在末行模式中输入m，ny
例如：3，5y复制第三行到第五行内容到缓存区。

使用p对复制的行进行粘贴

剪切 d+p

使用dd剪切一行
使用 行数n+dd 剪切n行
使用p对剪切的行进行粘贴

移动

Vim用数字+G来移动行，比如你要移动到342行，那就是342G

多行删除 d

首先在命令模式下，输入“:set nu”显示行号；
通过行号确定你要删除的行；
命令输入“:32,65d”,回车键，32-65行就被删除了

搜索字符串

:/字符串 可查找字符串
n:下一个匹配
N:上一个匹配

:set noh / :set nohlsearch ：关闭高亮

设置断点调试

（我们家晓燕教的）
两种方式：
方式一：让程序一行一行跑

python3 -m pdb wxy.py

方式二：设置断点
在代码要设置断点的地方加上

import pdb
pdb.set_trace()

之后正常运行程序即可

python wxy.py

调试开启后的命令：

(Pdb) l
会把代码打印出来，有>表示下一次要运行哪一行
(Pdb) n
让程序走一步
(Pdb) c
继续调试（运行到下一个断点）
(Pdb) p 变量名
显示该变量的值
ctrl+D
退出调试

撤销和反撤销

u 撤销上一步的操作
Ctrl+r 恢复上一步被撤销的操作

不需要冒号
