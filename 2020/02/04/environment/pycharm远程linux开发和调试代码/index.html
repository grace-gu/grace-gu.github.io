pycharm远程linux开发和调试代码
pycharm远程linux开发和调试代码和上一个不一样
PyCharm实现高效远程调试代码

过程

先打开本地项目，不打开的话以下配置在打开项目后失效

连接设置

Tools–Deployment–Configuration
选择SFTP

Connection
填写Host, User name和Password, root（默认的根目录）

advance查看是否勾选300S保持同步，需要勾选

Mapping
local path填写本地路径
Deployment path on server填写linux上基于root的代码路径。
比如：root设置为/data1/intern/GJ，则Deployment path应填写为/code/yolov3（代码真正路径为：/data1/intern/GJ/code/yolov3/）

此时Tools–Deployment–Browse Remote Host就可以看到你远程主机的信息

自动同步

Tools–Deployment–Configuration， 勾选Automatic Upload实现本地自动同步到远端

配置编译器

File -> Settings -> Project XXXX-> Project Interpreter
齿轮处点add, 选择SSH Interpreter, 填写Host, username, 下一步填写password
点选Interpreter的位置，这里说明一下，Linux服务器我使用的是conda装的Python，所以在conda的bin目录下有python, python3, python3.6，我选择的是python, 成功了
具体是/home/anaconda3/bin/python

选择这个编译器即可
