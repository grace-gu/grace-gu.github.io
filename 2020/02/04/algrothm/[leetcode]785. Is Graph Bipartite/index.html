[TOC]

题目

知识点

深度优先搜索DFS

深度优先搜索和广度优先搜索其实都是图形搜索算法
用几张图理解深度优先搜索
深度优先搜索用的是栈，即先进后出。进出原则：找出与此点邻接的且尚未遍历的点，进行标记，然后放入stack；如果此点没有尚未遍历的邻接点，则将此点从stack中弹出。

广度优先搜索BFS

广度优先搜索用的是栈，即先进先出。进出原则：找到一个点，把所有未被遍历的邻接点都放进queue；正常取出

引申：Dijkstra单源最短路径算法和Prim最小生成树算法都采用了和广度优先搜索类似的思想。

染色问题

说是染色，其实相当于给每个元素一个mark，标记它的状态。这里就用到了这个思想

Python all() any()

False: 0、’’、False或者空对象

any(x)
只有x里有一个是True, 就是True
只有x全为False, 才是False

all(x)
只要x里有一个是False, 就是False
只有x全为True,才是True
