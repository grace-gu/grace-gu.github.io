python中defaultdict方法的使用

在Python中，如果使用默认的dict，需要先判断某个元素是否存在，如果不存在直接操作会报错
这时候，就提出了defaultdict，使得进行defaultdict[key]或者dict.__getitem__(key)能不报错得到默认值。

defaultdict的使用方法：

  1. 初始化函数接受一个类型作为参数，当所访问的键不存在的时候，可以实例化一个值作为默认值。
  2. 该类除了接受类型名称作为初始化函数的参数之外，还可以使用任何不带参数的可调用函数，到时该函数的返回结果作为默认值。
